# Oppossum Towers

## Unity Version

2021.3.13f1

## Code Style

It is recommended to use EditorConfig to automatically adhere to a unified code style.

## Controls

| Keys       | Action        |
|------------|---------------|
| WASD       | Move          |
| Shift      | Sprint        |
| Ctrl       | Walk          |
| Space      | Jump          |
| E / LMB    | Interact      |
| T          | Tactical View |
| Mousewheel | Zoom          |
